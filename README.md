# Campaign Tracker

A simple web application to manage and track marketing campaigns with a clean dashboard, CRUD operations, and search/filter functionality.

# Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js/Express  
- **Database:** MongoDB   

# Features
- **Login Page:** Simple authentication (username & password).  
- **Dashboard:** Displays total, active, paused, and completed campaigns.  
- **CRUD Operations:**  
  - Add new campaigns  
  - Update campaign status (Active / Paused / Completed)  
  - Delete campaigns  
- **Search & Filter:** Filtering by campaign name, client, or status.  

## Setup Instructions

# Backend
1. Open terminal and navigate to backend folder:
cd backend

2. Install dependencies:
npm install

3. Start the server
npm start
Server runs at: http://localhost:5000
